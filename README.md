Adaptor between [LINE Messagaging API](https://developers.line.me/ja/reference/messaging-api/) and Discord Webhooks.

## Getting Started

```shell
export DISCORD_WEBHOOK_URL=<Your webhook url>
export LINE_CHANNEL_TOKEN=<Your token>

yarn install
node index.js
```
